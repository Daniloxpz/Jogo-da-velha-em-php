# Jogo da Velha em PHP 🎮

Este é um projeto acadêmico desenvolvido para a disciplina de **Programação para Computadores 1** do curso de Sistemas da informação.

## 📝 Sobre o Projeto
Um clássico Jogo da Velha (Tic-Tac-Toe) onde o usuário joga contra a máquina. O projeto utiliza a superglobal `$_SESSION` do PHP para persistir o estado do tabuleiro e gerenciar os turnos, vitórias e empates sem o uso de banco de dados.

## ⚙️ Funcionalidades
* Partida contra a máquina (jogadas aleatórias).
* Verificação automática de vitória (linhas, colunas e diagonais) e empate (velha).
* Persistência de dados utilizando sessões do PHP.
* Interface simples construída com HTML e CSS integrado.

## 🛠️ Tecnologias Utilizadas
* PHP
* HTML5
* CSS3
